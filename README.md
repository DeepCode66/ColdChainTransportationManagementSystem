## 功能介绍（完整见运行截图）
管理员：负责用户信息管理、订单信息管理、线路信息管理、车辆信息管理、物流信息管理和账单信息管理。能够添加新用户、搜索用户并进行编辑和管理操作。拥有查看和修改物流信息的权限。支持系统注册与登录管理功能。

用户：可以通过注册界面填写个人信息以完成注册，利用登录界面进行身份验证。用户能够查看和修改自己的个人信息，及查询订单状态。支持在物流信息管理中进行订单编号和状态的搜索。
